# Muta benchmark

## Requirement

- NodeJS >= 10

## Quick start

```shell
git clone https://github.com/homura/muta-benchmark.git
cd muta-benchmark
npm install
node index -d 5 http://127.0.0.1/graphql
```

or install globally

```
git clone https://github.com/homura/muta-benchmark.git
cd muta-benchmark
npm link
muta-bench -d 5 http://127.0.0.1/graphql
```
